# Web Research

## When to Use
- Enriching context with current information
- Verifying facts or data before making decisions
- Finding API documentation or technical references

## Best Practices
1. Use specific, targeted search queries
2. Cross-reference multiple sources
3. Prefer official documentation over third-party
4. Extract structured data (dates, numbers, names) explicitly
5. Cite sources in your output when relevant

## Pattern
1. Search with a focused query via WebSearch
2. Fetch the most relevant result with WebFetch
3. Extract the specific information needed
4. Summarize findings with source attribution
