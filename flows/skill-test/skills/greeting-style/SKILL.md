# Greeting Style

## When to Use
- Writing email introductions or sign-offs
- Formatting user-facing messages
- Adding a professional but friendly tone to outputs

## Templates

### Formal Greeting
"Dear [Name], I hope this message finds you well."

### Casual Greeting
"Hey [Name]! Quick update for you:"

### Sign-off
"Best regards, [Agent Name]"
