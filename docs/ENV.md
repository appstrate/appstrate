# Environment Variables

Primary source of truth: the `@appstrate/env` Zod schema (`packages/env/src/index.ts`) ‚Äî it wins on any conflict. This table documents the platform vars **plus a few sidecar-process vars** (`SIDECAR_*`, read in `runtime-pi/sidecar/`, NOT validated by `@appstrate/env`). It is neither a strict superset nor subset of the schema; consult the schema for the authoritative key set. Linked from `CLAUDE.md`.

`getEnv()` from `@appstrate/env` (Zod-validated, cached after first call, fail-fast at startup). Key variables:
